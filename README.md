## üöÄ Featured Projects

### üö¢ [AIMS (AI Marine Engineering System)](#)
*A Predictive and Preventative Maintenance Model for Marine Engineering.*
- **The Problem:** Optimizing maintenance schedules for marine diesel engines to prevent catastrophic failure.
- **The Solution:** An AI-powered system that uses **LightGBM/XGBoost** models to predict faults.
- **Key Tech:** Uses **SHAP** for explainability (so engineers understand *why* a fault is predicted), wrapped in a **FastAPI** backend with a **React** frontend.

### üîç [GlassBox AI](#)
*Glassbox AI analyzes government tenders in real-time using a multi-modal approach that examines both:*
- **Structured data**: Tender value, number of bidders, duration, complexity, PEP involvement
- **Unstructured text**: Tender descriptions analyzed with NLP to detect risk patterns
The system provides instant risk scoring with SHAP-based explainability and AI-generated insights powered by Google's Gemini API.

#### Key Features
